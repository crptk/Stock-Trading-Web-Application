This is one of the assignments I completed for CS50. It's basically a website that allows users to register accounts to buy/sell stocks using Yahoo's API.

Flask is used for the login system to check if a user is logged in, and what the program will do in response to it. It checks get and post requests, and navigates 
the user through the webpage accordingly.

SQL is the database used, it stores hashed passwords, usernames, balance amount, and transaction history.
